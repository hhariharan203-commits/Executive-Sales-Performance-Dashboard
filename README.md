# Executive Sales Performance Dashboard (Power BI)

## Project Overview
This project presents an executive-level sales performance dashboard built using Power BI to analyze business performance, profitability, and regional trends. The dashboard provides actionable insights to support strategic decision-making.

## Business Objectives
- Analyze total sales, total profit, and profit margin
- Identify best and worst performing product categories
- Evaluate regional sales and profit performance
- Detect loss-making sub-categories
- Monitor year-wise business growth trends

## Key Metrics
- Total Sales: $2.30M
- Total Profit: $286.40K
- Profit Margin: 12.47%

## Tools & Technologies Used
- Power BI
- SQL (Data validation and aggregation)
- Data Visualization
- Business Intelligence Techniques

## Dashboard Features
- KPI Cards for Sales, Profit, and Profit Margin
- Regional Sales & Profit Analysis
- Category-wise Performance Breakdown
- Loss-making Sub-category Identification
- Year-wise Sales Trend Analysis

## Files Included
- Executive_Sales_Performance_Analysis.pbix → Power BI dashboard file
- Dashboard_screenshot.png → Dashboard preview image

## Business Impact
This dashboard enables executives to identify profitability drivers, optimize pricing strategies, and improve business performance through data-driven insights.
